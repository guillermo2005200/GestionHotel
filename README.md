# GestionHotel

GestionHotel es un sistema de gestión hotelera de escritorio desarrollado en Java. Combina la potencia del framework Spring Boot para la lógica de negocio y la inyección de dependencias con JavaFX para la interfaz gráfica de usuario.

El objetivo del proyecto es facilitar la administración de reservas, clientes y habitaciones, proporcionando una herramienta robusta y escalable.

## Características

- Gestión de Habitaciones: Visualización de estado, alta, baja y modificación de habitaciones.
- Gestión de Clientes: Registro y almacenamiento de datos de huéspedes.
- Sistema de Reservas: Control de fechas de entrada y salida, asignación de habitaciones.
- Base de Datos Persistente: Almacenamiento seguro de toda la información mediante MySQL.

## Tecnologías Utilizadas

Este proyecto utiliza el siguiente stack tecnológico:

- **JavaFX**: Utilizado para el desarrollo de la interfaz gráfica de usuario (GUI), proporcionando una experiencia de escritorio moderna.
- **Spring Boot**: Framework principal utilizado para la gestión del contexto de la aplicación, inyección de dependencias y configuración automática.
- **MySQL**: Motor de base de datos relacional para el almacenamiento de la información.
- **JDBC**: Utilizado para la conectividad y ejecución de sentencias SQL directas contra la base de datos.
- **Lombok**: Biblioteca utilizada para reducir el código repetitivo (boilerplate) mediante anotaciones (Getters, Setters, Constructores, etc.).
